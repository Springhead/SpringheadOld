【アプリケーションテンプレートの使い方】

1. アプリケーションを新しく作る場所に、TemplateとInitApplication.exeを含むフォルダをまるごとコピーします。

2. フォルダの名前を、アプリケーション名にします。なお、これはそのままアプリケーションクラスの名前にもなるのでC++のクラス名に使える文字を使用のこと。

3. InitApplication.exeを実行します。


【アプリケーションテンプレートの作り方】

1. InitApplication.exeと同じ場所にTemplateフォルダを作り、もとになるファイルをその中に入れる必要があります。

2. ファイル名に_APPNAME_が含まれると、アプリケーション名で置き換わります。アプリケーション名とは、InitApplication.exe実行時にInitApplication.exeが入っているフォルダの名前です。

3. ファイル中の下記の文字列は置換されます。
− #{APPLICATIONNAME} : アプリケーション名
− #{SPRINGHEAD2} : アプリケーションを作成するフォルダからSPRINGHEAD2環境変数の値までの相対パス（ふつうslnファイルの中で使います）
