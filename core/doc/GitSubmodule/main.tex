\vspace{25pt}
<Springhead/buildtool>および<Springhead/dependency>は、
\TT{"sprphys/buildtool"}
および\TT{"sprphys/dependency"}をsubmoduleとして取り込んだものである。

\bigskip
\commands{%
    \ra git submodule add https://github.com/sprphys/buildtool.git buildtool\\
    \ra git submodule add https://github.com/sprphys/dependency.git dependency
}


\bigskip
\TT{"sprpyhs/buildtool"}または\TT{"sprphys/dependency"}で新たなcommit/pushを
行なっても\TT{"sprphys/Springhead"}のsubmoduleのリンクは自動的には追随せず、
次の作業を行なって正しいcommitをリンクし直さなければならない。

\medskip
なお、buildtoolまたはdependencyへの変更は必ず
\TT{"sprpyhs/buildtool"}または\TT{"sprphys/dependency"}で実施すること。
\TT{"sprphys/Springhead/buildtool"}および\TT{"sprphys/Springhead/dependency"}
で\TT{commit/push}した場合にどうなるかは未確認である。
