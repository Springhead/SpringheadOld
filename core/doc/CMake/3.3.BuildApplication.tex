\subsection{アプリケーションプログラムのビルド}
\label{subsec:BuildApplication}

\noindent
Springhead Libraryのビルドは、従来と変わりがありません。\\
\build へ移動し、\Path{Springhead.sln}を指定してVisual Studioを実行してください。

\medskip
``\ref{subsec:PrepareLibrary} 準備''でライブラリファイルのインストール先を
指定していなければ、\\
\hspace{20pt}\SprTop{/generated/lib/win64}\\
または\\
\hspace{20pt}\SprTop{/generated/lib/win32}\\
にライブラリファイルが生成されます。
